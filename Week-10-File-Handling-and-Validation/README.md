# Week 10 – File Handling and Validation

## Aim

Learn how to read and write text files and validate user input using Python.

---

## Programs

- validate-contact.py
- merge-text-files.py
- search-word-in-file.py

---

## Concepts Covered

- File Handling

---

## Learning Outcome

This week's programs introduced me to file operations in Python. I also learned how to validate user input and search for information inside text files.

## How to Run

```bash
python filename.py
```

Note: Some programs require sample `.txt` files. Keep those files in the same folder as the Python program.
